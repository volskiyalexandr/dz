Все довольно просто, ошибка была в том что 

firebase.auth().onAuthStateChanged
это подписка на событие которое произойдет неизвестно когда, и как только оно произойдет мы можем выполнить какие то действия, поэтому выносить как отдельный метод его не нужно и я добавил вот этот код вместо того что был

Это страница login.js

// Check Auth
firebase.auth().onAuthStateChanged(function(user) {
    if (user) window.location = 'home.html';
});
 
А это home.js

// Check Auth
firebase.auth().onAuthStateChanged(function(user) {
    if (!user) window.location = 'login.html';
});
 
Модуль User нам пригодится позже. 

 
Для создания проекта в firebase (этол не попало в видео)

1. перейти на страницу https://console.firebase.google.com/u/0/

2. Выбрать "Добавить проект"

3. Вводите название и нажимаете "Создать проект" 

Создание пользователя

1. Выбираем в меню Autentification

2. Выбираем пункт сверху "Способ входа" и включаем "Адрес электронной почты и пароль" как способ входа

3. Переходим в пункт "Пользователи" и добавляем нового пользователя.
